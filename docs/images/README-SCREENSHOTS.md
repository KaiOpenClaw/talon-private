## Screenshot Guidelines

### Technical Requirements:
- **Browser**: Chrome with clean UI (hide bookmarks, extensions)
- **Resolution**: 2x retina for crisp appearance
- **Zoom Level**: 100% for accurate representation
- **Theme**: Dark mode (matches developer preference)

### Content Guidelines:
- **Real Data**: Use actual agent names and realistic content
- **Clean State**: No error messages or loading states
- **Professional**: Remove any test/debug content
- **Consistent**: Same agents/data across screenshots

### Optimization:
- **Compression**: Use TinyPNG for size optimization
- **Format**: PNG for screenshots, WebP for modern browsers
- **Loading**: Provide low-res placeholder for progressive loading
- **Accessibility**: Include descriptive alt text
