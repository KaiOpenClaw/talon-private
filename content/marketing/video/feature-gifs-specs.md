## Technical Recording Specifications

### Screen Setup
- Resolution: 1270x760px exactly
- Dark theme enabled
- Use real OpenClaw data
- Clean desktop background
- Hide browser UI elements

### Recording Settings
- Frame rate: 24 FPS (optimal for GIF)
- Smooth cursor movement
- Intentional interactions (no accidental clicks)
- Consistent timing between actions

### GIF Optimization
- Use tools like Gifski or GIMP for optimization
- Reduce color palette for smaller file size
- Maintain visual quality under 10MB
- Loop seamlessly

### Post-Processing
- Add subtle drop shadows if needed
- Ensure text is readable
- Optimize for social media sharing
- Test on different devices

## Content Guidelines

### Visual Style
- Consistent with brand colors
- Professional, clean interface
- Dark theme throughout
- High contrast for readability

### User Experience
- Natural, realistic interactions
- Appropriate pacing (not too fast/slow)
- Clear cause-and-effect relationships
- Professional cursor movement

### Quality Standards
- No visual glitches or artifacts
- Smooth animations
- Clear text rendering
- Consistent visual hierarchy

### Social Media Formats

### File Management
