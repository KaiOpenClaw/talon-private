### Technical Requirements

#### Hardware & Software
- **High-resolution displays**: 2560x1440 minimum for retina quality
- **Professional recording tools**: Snagit, CleanShot, or Loom for GIFs
- **Design software**: Figma/Sketch for social media graphics
- **Browser setup**: Chrome/Firefox with developer tools hidden
- **Screen recording**: 60fps capability for smooth motion capture

#### Environment Preparation  
- **Clean Talon instance** with professional sample data
- **Realistic agent configurations** (no test-agent-123 naming)
- **Professional conversation content** with meaningful technical discussions
- **Consistent UI state** across all screenshots for brand consistency
- **Mobile testing devices** for responsive design verification

### Content Standards

#### Screenshot Quality
- **Resolution**: 2560x1440 (retina quality, web-optimized to <500KB)
- **Format**: PNG with transparency support for overlays
- **Compression**: Optimized without visible quality loss
- **Color accuracy**: Consistent brand colors across all assets
- **Professional presentation**: Clean, organized interface states

#### GIF/Video Quality
- **Resolution**: 1920x1080 for demos, 1200x675 for social media
- **Duration**: 10-25 seconds for optimal engagement
- **Frame rate**: 30fps for smooth interactions, 60fps for recording
- **File size**: <2MB for GIFs, <10MB for videos
- **Smooth motion**: Natural pacing with strategic pauses
