# Blog Assets

This directory contains visual assets for blog posts:

- talon-dashboard-hero.png - Main dashboard screenshot for hero sections
- talon-login.png - Login screen demonstration  
- talon-workspace-example.png - Agent workspace interface
- talon-search-demo.gif - Semantic search in action
- talon-cron-dashboard.png - Cron job management interface

## Image Guidelines

- **Resolution**: 1920x1080 for hero images, 1200x800 for inline screenshots
- **Format**: PNG for UI screenshots, GIF for animated demos, JPG for photos  
- **Optimization**: Use WebP when possible for better performance
- **Alt Text**: Always include descriptive alt text in markdown

## Capturing Screenshots

For consistent branding:
1. Use dark mode for all screenshots
2. Ensure clean demo data (no personal information)  
3. Highlight key features with arrows or callouts when needed
4. Maintain consistent browser/viewport size
